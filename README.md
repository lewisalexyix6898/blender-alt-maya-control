## What is Alt Maya Blender Ducktape?

Alt Maya Blender Ducktape is a Blender plugin for users who want a more familiar viewport navigation scheme without sacrificing Blender's native keymaps. While Alt is held down, the add-on shifts navigation into an industry-compatible mode, making orbit control behave more like Maya.

This is a practical fit for artists and technical users who move between Blender and other 3D or compositing tools like Maya, Houdini, Unreal Engine, Unity, Cinema 4D, Modo, Substance Painter, Substance Designer, Fusion, Mari, Clarisse iFX, and Nuke. The intent is to reduce context switching while leaving the default Blender experience available for tutorials and everyday work.

---

## What it does

- Temporarily activates an industry-compatible keymap while Alt is held
- Restores Blender's default keymaps as soon as Alt is released
- Tones viewport orbiting to feel closer to Maya-style navigation
- Makes it easier to switch between Blender and other 3D applications
- Leaves the usual Blender workflow intact for learning and reference
- Supports a navigation pattern that feels familiar to users of major DCC tools
- Concentrates on viewport orbit and navigation behavior only
- Uses a lightweight add-on approach that coexists with Blender's input system

## Using the add-on

Once the add-on is enabled, hold Alt while navigating in the viewport to temporarily switch into the industry-compatible navigation mode.

Typical workflow:

- Open a 3D view in Blender
- Hold Alt
- Orbit, pan, or navigate as you normally would with a Maya-like setup
- Release Alt to return to Blender's default controls

This helps you stay productive in Blender without having to relearn your muscle memory for every task.

## FAQ

### Does this replace Blender's normal controls?
No. The plugin keeps Blender's default keymaps in place whenever Alt is not being used.

### When does the Maya-like behavior activate?
It becomes active while Alt is held, then switches back to the regular Blender setup after release.

### Who is this for?
It is a good match for Blender users who are accustomed to navigation in Maya or other major 3D applications with similar orbit controls.

### Where are updates posted?
Check the repository and the download link at the top of this README for the latest build and project changes.

### What should I do if the navigation does not change?
Confirm that the add-on is enabled in Blender, then review your keymap settings and restart Blender if needed.
